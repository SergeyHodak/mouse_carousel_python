чтобы довавить новый репозиторий на гитхабе
1) создать на компе папочку, открыть ее в пайчарме, и в Terminal пайчарма проверить что вы в этом каталоге
2) git init  #Сначала создайте новый репозиторий в этом каталоге
3) git add . #добавить все файлы в свой репозиторий
4) git commit -m "start" #зафиксировать изменения в комит, чтобы git мог их отслеживать
5) создать новый репозиторий на гитхабе
6) git remote add origin https://github.com/SergeyHodak/mouse_carousel_python.git  #ссылка с репозитория на гитхабе
7) git push -u origin master # выгрузить на гитхаб

Дальнейшие добавлания изменений
git init ### запускает работу с Гитом
    git add . ### добавляет все файлы проэкта в репозиторий гит
    git commit -m "start" ### создаст коммит с "именем"
    git push ### загружает все измененя сделанные в гит на гитхаб
